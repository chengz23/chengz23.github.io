4. tmux 常用命令汇总表

操作 命令 / 快捷键
创建新会话 tmux new -s <name>
断开当前会话 Ctrl + b 然后 d
查看会话列表 tmux ls
恢复指定会话 tmux attach -t <name>
重命名会话 tmux rename-session -t old new
创建新窗口 Ctrl + b 然后 c
切换窗口 Ctrl + b 然后 数字键 / n / p
分割垂直窗格 Ctrl + b 然后 "
分割水平窗格 Ctrl + b 然后 %
窗格间切换 Ctrl + b 然后 方向键
关闭当前窗格 在窗格中运行 exit 或 Ctrl + d
滚动历史输出 Ctrl + b 然后 [，再用方向键滚动
复制模式 Ctrl + b 然后 [，使用 Space 选中，再按 Enter 复制
粘贴 Ctrl + b 然后 ]

✨ 总结

tmux 是远程开发过程中不可或缺的工具之一，它相比 nohup 拥有更丰富的交互性和可控性，尤其适合科研、数据挖掘等需长时间训练和调试的场景。
